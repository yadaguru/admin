Internal name for reference (does not appear in UI) max 28 characters. Displayed in generated SMS messages. Internal category for organization (does not appear in UI). Categories can be edited here. Ctrl/Cmd click to select multiple timeframes. Timeframes can be edited here. The following variables are available on all rich-text fields on this form:
  • %APPLICATION_DATE% - date the application is due
  • %REMINDER_DATE% - date of the current reminder
  • %SCHOOL% - name of school
 NOTE: Late message/detail displays when reminders are overdue. If no message/detail is specified, the default message/detail will be used.
